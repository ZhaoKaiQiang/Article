# 如何提高版本质量？

### 日常数据检测

1. 对产品的基础数据的检测
2. 对线上版本的稳定性相关指标的检测
3. 服务器接口的每日自动化测试
4. 线上服务实时报警机制
5. 用户反馈的及时跟进

### 持续集成

好处主要有以下几点：

1. 及时暴露问题
2. 快速提供产品
3. 减少重复劳动
4. 提高项目可见性，增强团队信心

持续集成通常会挂接以下任务：

1. 编译打包，失败报警
2. 静态代码扫描日报
3. 包检查（大小、冗余文件）
4. 自动化测试
5. 流量测试、电量测试
6. 性能测试

### 内部测试

1. 研发自测
2. 设计走查
3. 产品走查
